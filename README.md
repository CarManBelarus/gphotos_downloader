# Спампоўшчык агульных альбомаў Google Фота 🚀

Асінхронны высокапрадукцыйны скрыпт на Python для масавага спампоўвання фота і відэа з агульных альбомаў Google Фота праз публічную спасылку. Без аўтарызацыі, без страты якасці.

## 🌟 Асаблівасці
- Поўны абыход лімітаў Google: Скрыпт выкарыстоўвае рэверс-інжынірынг схаванага RPC API (/batchexecute), што дазваляе спампоўваць велізарныя альбомы (тысячы файлаў), абыходзячы стандартнае абмежаванне Google у 300 файлаў на старонку.
- Максімальная якасць: Усе медыя спампоўваюцца ў арыгінальнай якасці (выкарыстоўваюцца мадыфікатары =d і =dv).
- Разумная праца з відэа: Скрыпт "на хаду" адрознівае фатаграфіі ад цяжкіх .MP4/.MTS відэаролікаў і карэктна выцягвае арыгіналы відэа, а не іх прэв'ю.
- Асінхроннасць: Напісаны на aiohttp з пулам злучэнняў (Keep-Alive). Спампоўвае файлы ў некалькі патокаў, забіваючы ваш інтэрнэт-канал на 100%.
- Патокавы запіс (Chunked Streaming): Спажывае мінімум аператыўнай памяці (менш за 50 МБ), нават калі пампуе відэа памерам 10 гігабайт.
- Інфарматыўны прагрэс-бар: Адлюстроўвае колькасць файлаў і бягучую хуткасць спампоўвання (МБ/с) з дапамогай tqdm.
- Адмоваўстойлівасць: Механізм Exponential Backoff аўтаматычна паўтарае спробы спампоўвання ў выпадку абрыву сеткі.

## 📦 Як спампаваць праграму і карыстацца ёй (Для карыстальнікаў Windows)

Вам неабавязкова ўсталёўваць Python і разбірацца з кодам. Вы можаце спампаваць ужо гатовую праграму, якая працуе цалкам аўтаномна:

1. Перайдзіце ў раздзел [Releases](https://github.com/CarManBelarus/gphotos_downloader/releases) (Рэлізы) справа на гэтай старонцы.
2. Спампуйце апошнюю версію файла gphotos_downloader.exe.
3. Запусціце спампаваны файл двайным пстрыкам.
4. У акне, якое з'явіцца, устаўце публічную спасылку на альбом (кшталту https://photos.app.goo.gl/...) і пазначце назву папкі для захавання. 
5. Дачакайцеся завяршэння — файлы з'явяцца ў той жа папцы, дзе ляжыць скрыпт!

*(Калі Абаронца Windows (SmartScreen) выдасць папярэджанне "Невядомы выдавец", націсніце "Падрабязней" -> "Выканаць у любым разе").*

## 🛠 Запуск з зыходнага коду (Для распрацоўшчыкаў / Linux / macOS)

Калі вы хочаце запусціць скрыпт наўпрост праз Python:

1. Усталюйце залежнасці:
pip install aiohttp aiofiles tqdm
2. Запусціце скрыпт:
python gphotos_downloader.py
